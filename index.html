Features

Coded in Java

The bot is completely coded in Java utilizing the Discord JDA API and the top.gg API for updating the server count on their page.

Free forever

The bot will stay completely free as I know the struggle to find a good working bot that is also free to use like NukeBot is.

100% safe

The bot doesn't collect any data except for the amount of servers it is in.
You'll see that in it's status where it shows how many server owners trust my NukeBot.

Active support

I provide different possibilities to contact me for support or improvement, either via Discord or using Twitter to contact the support account provided on this page.
